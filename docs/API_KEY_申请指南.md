## 一、OpenAI API Key（必须）

**用途**：负责"看懂"你的产品图片、生成视频脚本、生成白底产品图，是整个系统的核心大脑。

> ⚠️ OpenAI 在中国大陆无法直接访问，需要使用科学上网工具（VPN）完成注册和充值。

### 申请步骤

**第 1 步：打开网站**

用浏览器访问：https://platform.openai.com

如果打不开，请先开启 VPN，节点选择美国或日本。

---

**第 2 步：注册账号**

点击页面上的 **Sign up**（注册）。

- 可以用 **Google 账号**直接登录（最简单）
- 也可以用邮箱注册，注册后需要验证邮箱
- 手机号验证：需要填写手机号，**中国大陆号码（+86）通常可以使用**

如果已有 ChatGPT 账号，直接点 **Log in** 登录，用同一个账号即可。

---

**第 3 步：进入 API Key 管理页面**

登录后，访问：https://platform.openai.com/api-keys

或在左侧菜单点击 **API keys**。

---

**第 4 步：创建 API Key**

1. 点击右上角 **+ Create new secret key**
2. 在弹出窗口的 Name 里填写一个名字，比如 `video-generator`（方便自己识别）
3. 点击 **Create secret key**
4. 屏幕上会出现一串以 `sk-proj-` 开头的字符，**立刻复制保存**

> ⚠️ 这个 Key **只显示一次**，关闭弹窗后就看不到了。请粘贴到备忘录或微信文件传输助手保存好。

---

**第 5 步：充值**

API Key 申请好后还需要充值才能使用。

1. 点击左侧菜单 **Billing**（账单）
2. 点击 **Add payment details**，绑定一张信用卡（支持 Visa / Mastercard）
   - 国内双币信用卡一般可以使用，如招行、建行等的 Visa 卡
3. 绑卡后点击 **Add to credit balance**，充值金额
   - 建议首次充值 **$10**（约 73 元人民币），够生成约 90 个视频

---

**第 6 步：把 Key 发给开发者**

将复制的 Key（`sk-proj-xxxxxx`）通过微信发给开发者即可。

## 二、豆包 Seedance API Key（推荐）

**用途**：负责根据产品图生成视频，生成 5 秒 9:16 竖版短视频，单次约 ¥0.9，国内直连无需 VPN。

### 申请步骤

**第 1 步：注册火山引擎账号**

用浏览器访问：https://www.volcengine.com

点击右上角 **注册**，用手机号注册即可（中国大陆手机号直接可用）。

如果已有抖音 / 飞书账号，可以扫码直接登录，更方便。

---

**第 2 步：进入火山方舟控制台**

登录后，访问：https://console.volcengine.com/ark

或在首页产品列表搜索 **火山方舟**，点击进入。

首次进入可能需要点击 **开通服务**，按提示操作即可，不需要额外费用。

---

**第 3 步：开通 Seedance 模型**

1. 在火山方舟控制台左侧菜单找到 **模型广场** 或 **在线推理**
2. 搜索 `Seedance`，找到 **doubao-seedance-1-0-lite-i2v**
3. 点击 **开通**（首次使用需要开通，免费操作）

---

**第 4 步：创建 API Key**

1. 在控制台左侧菜单点击 **API Key 管理**
2. 点击 **新建 API Key**
3. 填写名称（如 `video-generator`），点击确认
4. 复制生成的 API Key，**立刻保存到备忘录**

> ⚠️ Key 同样只显示一次，请及时保存。

---

**第 5 步：充值**

1. 点击右上角账户头像 → **费用中心** 或 **充值**
2. 选择充值金额，支持支付宝 / 微信支付
   - 建议首次充值 **¥50**，够生成约 55 个视频

---

**第 6 步：把 Key 发给开发者**

将复制的 API Key 通过微信发给开发者即可。

## 三、Creatok API Key（备选）

**用途**：与豆包 Seedance 功能相同，负责生成视频，生成 15 秒视频，单次约 ¥2.2。如果已经申请了豆包 Seedance，这个可以不申请。

> ⚠️ Creatok 是境外服务，需要使用 VPN 访问。

### 申请步骤

**第 1 步：打开网站**

开启 VPN 后，用浏览器访问：https://www.creatok.ai

---

**第 2 步：注册账号**

点击 **Sign up** 或 **Get Started**，用邮箱注册。注册后验证邮箱。

---

**第 3 步：进入 API Key 页面**

登录后，在右上角点击头像或进入 **Dashboard**（控制台），找到以下任一入口：

- **API Keys**
- **Developer**
- **Settings → API**

---

**第 4 步：创建并复制 API Key**

点击 **Create API Key** 或 **Generate Key**，复制生成的 Key，保存好。

---

**第 5 步：充值**

在账户设置或 Billing 页面充值，支持国际信用卡（Visa / Mastercard）。

- 建议首次充值 **$20**，够生成约 60 个视频

---

**第 6 步：把 Key 发给开发者**

将复制的 API Key 通过微信发给开发者即可。

## 五、常见问题

**Q：API Key 是什么？安全吗？**

API Key 类似一个"通行证"，系统用它来调用 AI 服务。Key 只保存在您自己的服务器上，不会上传到其他地方。建议不要把 Key 截图发到公开群里，一对一发给开发者即可。

**Q：OpenAI 网站打不开怎么办？**

需要使用 VPN，节点选择美国、日本或新加坡。推荐在电脑上操作，手机也可以但稍麻烦。

**Q：没有境外信用卡，OpenAI 怎么充值？**

可以考虑：
- 使用国内双币信用卡（招行、建行等带 Visa 标的卡）
- 请有境外卡的朋友代充，转账给他们

**Q：Key 只显示一次，忘记保存了怎么办？**

删掉这个 Key，重新创建一个新的即可，不影响使用。

**Q：先充多少合适？**

测试阶段建议按上表最低金额充值，系统测试通过、效果满意后再按实际用量补充。
